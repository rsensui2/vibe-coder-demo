# ソリューション概要

## プロダクト構成

### コア機能: 予算ベース車格比較エンジン

```
[ユーザー入力]
  予算: 400万円
  希望ボディタイプ: セダン / SUV / ALL
  優先項目: 走り > 見た目 > 燃費
       ↓
[CarScale エンジン]
  ├── 新車データベース検索
  ├── 中古車データベース検索（年式・走行距離フィルタ）
  ├── 車格スコア算出
  ├── 維持費シミュレーション
  └── 総合スコアリング
       ↓
[出力: 比較ダッシュボード]
```

### 比較ダッシュボード画面構成

#### メインビュー: 2カラム比較
| | 新車 | 中古車 |
|---|------|--------|
| **例** | トヨタ カムリ WS 2026年式 | メルセデス E300 2023年式 |
| **価格** | ¥3,980,000 | ¥3,950,000 |
| **車格スコア** | ★★★☆☆ (3.0) | ★★★★★ (5.0) |
| **エンジン** | 2.5L 直4 HV 218ps | 2.0L 直4 ターボ 258ps |
| **全長** | 4,885mm | 4,940mm |
| **装備** | Toyota Safety Sense | AMGライン、Burmester |
| **年間維持費** | ¥280,000 | ¥420,000 |
| **3年後残価率** | 55% | 45% |

#### サブビュー
1. **レーダーチャート**: 走行性能、快適性、安全性、燃費、ブランド力を5軸で比較
2. **維持費タイムライン**: 3年/5年/7年の総コスト推移グラフ
3. **類似車リスト**: 予算±50万円の代替候補

### 技術スタック
- **フロントエンド**: Next.js + Tailwind CSS
- **バックエンド**: Supabase (PostgreSQL + Auth + Edge Functions)
- **車両データ**: カーセンサーAPI / スクレイピング + 独自DB
- **AI**: 車格スコア算出、レコメンドエンジン（OpenAI API）
- **ホスティング**: Vercel

### MVP（最小実装）
1. 予算入力フォーム
2. 新車5台 vs 中古車5台のリスト表示
3. 車格スコア（スペックベースの簡易算出）
4. 基本スペック比較テーブル

→ MVPは **2週間** で構築可能（Vibe Coding前提）

# ビジネスモデル — TaxViz（タックスビズ）

## 収益構造

### プラン設計

| プラン | 月額 | 年額 | 対象 |
|--------|------|------|------|
| **Free** | ¥0 | ¥0 | 給与所得のみ（医療費控除・ふるさと納税） |
| **Standard** | ¥980 | ¥9,800 | 副業・フリーランス（事業所得・不動産所得） |
| **Pro** | ¥2,980 | ¥29,800 | 複数事業・消費税申告・法人成り検討 |
| **Tax Advisor** | ¥9,800 | ¥98,000 | 税理士向けクライアント管理・一括処理 |

### 収益モデル

```
[Free] ──→ 確定申告体験 ──→ [Standard/Pro] 有料転換
   │                              │
   ├── 広告収益（税理士マッチング）    ├── サブスクリプション
   │                              │
   └── データインサイト販売（匿名化）  └── e-Tax連携プレミアム
```

## KPI設計

### North Star Metric
**「入力完了→e-Tax提出までの完走率」**

### ファネルKPI

```
認知（広告/SNS） → 登録 → 入力開始 → 入力完了 → e-Tax提出 → 翌年継続
  100%           30%     60%       45%       80%       70%
```

### 重要KPI

| KPI | 定義 | 目標 |
|-----|------|------|
| Time to First Value | 登録→初回税額表示まで | 3分以内 |
| 入力完走率 | 入力開始→全項目完了 | 45% |
| 7日リテンション | 登録後7日以内の再訪問 | 60% |
| 有料転換率 | Free→Standard/Pro | 5%（1年目） |
| ARPU | 有料ユーザー平均単価 | ¥1,800/月 |
| CAC | 顧客獲得コスト | ¥3,000以下 |
| LTV/CAC | | 3.0以上 |

## 市場規模

### TAM/SAM/SOM

| 指標 | 規模 | 算出根拠 |
|------|------|----------|
| **TAM** | 3,600億円 | 確定申告対象者2,400万人 × 平均年間15,000円 |
| **SAM** | 720億円 | 自分で申告する層（約480万人）× 15,000円 |
| **SOM** | 15億円 | SAMの2%獲得（3年目目標） |

### 市場環境

- 確定申告ソフト市場: 年間成長率12%（2023-2028予測）
- e-Tax利用率: 65%超（2025年）→ デジタル化加速
- 副業解禁: 副業人口は2020年比で2.3倍に増加
- インボイス制度: 免税事業者の申告ニーズ急増

## 競合優位性の持続

### 参入障壁

1. **リアルタイム税額計算エンジン** — 複雑な税法ロジックの実装に6ヶ月以上
2. **UXノウハウ** — 確定申告特化の入力体験は既存プレイヤーには後付けが困難
3. **ユーザーデータの蓄積** — パーソナライズされた節税提案の精度向上
4. **税理士ネットワーク** — マッチングプラットフォームとしての両面市場効果

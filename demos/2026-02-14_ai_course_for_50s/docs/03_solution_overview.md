## 画面設計

### 1. 受講生向けアプリ（PWA / スマホ最適化）

#### ホーム画面
- 今日のレッスン（カード形式、大きなタップ領域）
- 学習進捗バー（「あと3レッスンで初級マスター！」）
- 今週のAI活用ヒント
- 次回ライブ講座までのカウントダウン

#### レッスン画面
- 動画プレーヤー（字幕付き、再生速度0.5x〜2x）
- ステップバイステップ操作ガイド（スクリーンショット付き）
- 「やってみよう」ボタン → AIチャット画面に直接遷移
- つまづきポイントFAQ（折りたたみ式）

#### AIお試し画面
- 事前設定済みのプロンプトテンプレート（タップで入力）
  - 「旅行プランを作って」
  - 「この薬の副作用を教えて」
  - 「孫の誕生日プレゼントを提案して」
- 入力支援：音声入力ボタン、定型文ボタン
- 結果のコピー・LINE共有ボタン

#### マイページ
- 修了バッジコレクション
- 学習カレンダー（連続学習日数）
- お気に入りプロンプト保存
- 受講プラン変更

### 2. 講師向け管理画面

- 受講生一覧（進捗・最終ログイン・つまづきポイント）
- ライブ講座スケジュール管理
- 質問一覧（LINE連携）と回答テンプレート
- コンテンツ管理（レッスン追加・編集）
- 売上・KPIダッシュボード

### 3. 法人管理画面

- 社員の受講状況一括管理
- 部署別進捗レポート
- 修了証PDF発行
- 請求書・契約管理

## カリキュラム設計

### 全6ステージ構成（各4週間）

| ステージ | テーマ | ゴール |
|---------|--------|--------|
| Stage 1 | AIってなに？ | ChatGPTに質問して答えを得られる |
| Stage 2 | もっと上手に聞こう | プロンプトの基本（具体的に聞く、条件をつける） |
| Stage 3 | 生活に活かそう | 旅行・料理・健康・お金の相談をAIでこなす |
| Stage 4 | 画像も音声も | 画像生成AI、音声入力、翻訳AI活用 |
| Stage 5 | AIを味方にしよう | 定型作業の自動化、AIアシスタント設定 |
| Stage 6 | 人に教えられるレベルへ | 学んだことを他の人に教えるワークショップ |

### 1レッスンの構成（60分）

```
00:00-05:00  前回の復習クイズ（3問）
05:00-15:00  ミニ講義（動画 or ライブ）
15:00-35:00  ハンズオン（実際にAIを使う）
35:00-50:00  グループシェア（やってみた結果を共有）
50:00-55:00  今週のチャレンジ発表
55:00-60:00  質疑応答
```

## 技術スタック

### フロントエンド
| 技術 | 用途 | 選定理由 |
|------|------|---------|
| Next.js 15 | Webアプリフレームワーク | SSR/SSG対応、高速表示 |
| Tailwind CSS | UIスタイリング | レスポンシブ対応が容易 |
| PWA | モバイルアプリ化 | インストール不要、ホーム画面追加 |
| Radix UI | アクセシブルコンポーネント | シニア向けa11y対応 |

### バックエンド
| 技術 | 用途 | 選定理由 |
|------|------|---------|
| Supabase | DB・認証・ストレージ | PostgreSQL + Auth + Storage一体型 |
| Vercel | ホスティング | Next.jsとの親和性、Edge Functions |
| Stripe | 決済 | サブスクリプション管理、日本円対応 |
| LINE Messaging API | コミュニティ連携 | 50代以上のLINE利用率95%以上 |

### AI連携
| サービス | 用途 |
|---------|------|
| OpenAI API (GPT-4o) | メイン講座用AIチャット |
| Anthropic Claude | 補助AI、比較体験 |
| Google Gemini | マルチモーダル体験（画像認識等） |
| Whisper API | 音声入力→テキスト変換 |

### インフラ
| サービス | 用途 |
|---------|------|
| Vercel | フロントエンドデプロイ |
| Supabase Cloud | バックエンド |
| Cloudflare | CDN・DDoS対策 |
| Sentry | エラー監視 |
| Mixpanel | ユーザー行動分析 |

## MVP（最小実用プロダクト）

### MVP Phase 1（開発期間: 4週間）

**含むもの:**
1. LP（ランディングページ）
2. 体験講座申込フォーム
3. Stage 1の動画コンテンツ（4レッスン分）
4. AIお試し画面（プロンプトテンプレート3種）
5. LINE公式アカウント（質問受付）
6. Stripe決済（ベーシックプランのみ）

**含まないもの:**
- 講師管理画面（Notionで代替）
- 法人向け機能
- バッジ・ゲーミフィケーション
- 動画アーカイブ検索

### MVP Phase 2（MVP1後4週間）
- 全6ステージのコンテンツ
- プレミアム・マスタープラン
- 講師管理画面
- Zoom連携（ライブ講座）

### MVP Phase 3（さらに4週間後）
- 法人管理画面
- バッジ・修了証システム
- 講師養成プログラム
- 自治体向けライセンス販売

## アクセシビリティ方針

| 項目 | 対応 |
|------|------|
| 文字サイズ | 最小16px、推奨18px。拡大ボタン常時表示 |
| コントラスト | WCAG AAA準拠（7:1以上） |
| タップ領域 | 最小48px × 48px |
| 音声対応 | 全画面で音声入力対応、読み上げ対応 |
| 動画字幕 | 全動画に日本語字幕必須 |
| 操作ステップ | 1タスク最大3タップ以内 |
| エラー表示 | 大きな赤文字 + 具体的な修正案 |
| ナビゲーション | パンくずリスト + 「戻る」ボタン常時表示 |

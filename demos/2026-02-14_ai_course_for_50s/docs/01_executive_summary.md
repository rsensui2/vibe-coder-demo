# エグゼクティブサマリー：50代からのAI講座

## コンセプト

**「スマホで文字が打てれば、AIは使える」**

50代以上のスマートフォンユーザーを対象に、AI（ChatGPT、Gemini、Claude等）の活用スキルを段階的に習得できるオンライン＋オフライン講座サービス。プログラミング不要、特別なPCスキル不要。「スマホで文字を打つ」というたった一つのスキルさえあれば、AIの恩恵を最大限に受けられることを体験ベースで学ぶ。

## 解決する課題

| 課題 | 詳細 |
|------|------|
| AI格差の拡大 | 若年層とシニア層のAI活用率に大きな格差。総務省調査（2025）では60代のChatGPT利用率は8.2%、20代は47.3% |
| 情報弱者化リスク | AI非活用者は今後の行政手続き・医療情報・金融サービスでも不利になるリスク |
| 既存講座のミスマッチ | 現在のAI講座はエンジニア向けかビジネスパーソン向けが主流。50代以上の生活者視点の講座はほぼ皆無 |
| 学習への心理的障壁 | 「AIは難しそう」「自分には無理」という先入観。実際は文字入力さえできれば十分 |
| 孤立した学習環境 | 独学では挫折率が高い。仲間と一緒に学ぶコミュニティの欠如 |

## ターゲットユーザー

### プライマリターゲット
- **年齢:** 50〜70代
- **デジタルスキル:** スマホでLINE・メールができるレベル
- **動機:** 「AIって何？」から「仕事や生活に使いたい」まで幅広い
- **推定人口:** 約3,200万人（50〜70代のスマホ利用者、総務省通信利用動向調査2024）

### セカンダリターゲット
- **企業の人事・研修担当者:** シニア社員向けAIリスキリング研修として導入
- **自治体・公民館:** デジタルデバイド解消施策として採用
- **家族:** 親にAIを教えたいが自分では教えられない30〜40代

## 差別化ポイント

### 1. 「スマホファースト」設計
PCを前提としない。受講者の98%が持っているスマートフォンだけで完結する講座設計。

### 2. 「生活密着型」カリキュラム
ビジネス用途ではなく、日常生活での活用に焦点：
- 旅行プランの作成
- 健康相談の壁打ち
- 孫へのプレゼント選び
- 料理レシピのアレンジ
- 終活・相続の情報整理

### 3. 「伴走型」サポート体制
- 週1回のオンラインライブ講座（Zoom）
- LINEグループでの質問し放題
- 月1回の対面交流会
- 「AIバディ」制度（受講生同士のペアリング）

### 4. 「成功体験ファースト」の教育哲学
初回30分でAIに質問→回答を得る体験を必ず成功させる。理論より実感。

### 5. シニア特化のUI/UX
- 大きな文字サイズ
- ステップ数を最小化したナビゲーション
- 音声入力対応ガイド
- 紙のワークシート併用

## 市場機会

| 指標 | 数値 |
|------|------|
| 国内50代以上のスマホ保有率 | 82.7%（2024年、総務省） |
| 生涯学習市場規模 | 約2.5兆円（2024年、矢野経済研究所） |
| シニア向けデジタル教育市場 | 約1,200億円（2025年推定） |
| ChatGPT認知率（60代） | 71.3%（認知はあるが使えない層が大半） |
| リスキリング補助金対象 | 経産省「リスキリングを通じたキャリアアップ支援事業」適用可能性あり |

## ビジョン

**「AIが使えないから不便」ではなく「AIがあるから楽しい」と思える50代以上を100万人つくる。**

2027年までに受講者1万人、2030年までにシニアAIリテラシーの標準プラットフォームとなることを目指す。

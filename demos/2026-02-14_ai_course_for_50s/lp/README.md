# 50代からのAI講座 - ランディングページ（3パターン）

## 📋 プロジェクト概要

「50代からのAI講座」のランディングページ3パターンを作成しました。

**サービス概要:**
- スマホで文字しか打てない人のためのAI活用講座
- ターゲット: 50〜70代のスマホユーザー
- 月額制（¥4,980〜¥19,800）
- 週1ライブ講座 + LINEコミュニティ + 対面交流会

---

## 🎨 3つのデザインパターン

### Pattern 1: Modern（モダン）
**ファイル:** `pattern1_modern.html`

**デザインコンセプト:**
- クリーンでシンプルなモダンデザイン
- v0/Vercel風の洗練されたUI
- 青系のアクセントカラー（#0070f3）
- 白ベース + グラデーション背景

**特徴:**
- ミニマルで見やすいレイアウト
- カードベースのデザイン
- ホバーアニメーションで洗練された印象
- プロフェッショナルな雰囲気

**こんな方におすすめ:**
- 信頼感と安定感を重視したい
- シンプルでわかりやすいデザインが好み
- IT系に抵抗が少ない層

---

### Pattern 2: Premium（プレミアム）
**ファイル:** `pattern2_premium.html`

**デザインコンセプト:**
- 高級感のある上質なデザイン
- 明朝体フォントで品格を演出
- ゴールド系のアクセント（#daa520）
- 黒 + ベージュ系の配色

**特徴:**
- ラグジュアリーな雰囲気
- 細部までこだわった装飾
- スライドアニメーションと光沢エフェクト
- 大人向けの落ち着いたトーン

**こんな方におすすめ:**
- 質の高いサービスを求める層
- 価格よりも価値を重視
- ゆとりがあり、上質な体験を求める50〜70代

---

### Pattern 3: Casual（カジュアル）
**ファイル:** `pattern3_casual.html`

**デザインコンセプト:**
- 親しみやすく温かみのあるデザイン
- 赤系の明るいアクセント（#ff6b6b）
- 絵文字を多用した楽しい雰囲気
- 丸みを帯びたフレンドリーなUI

**特徴:**
- 「一緒に楽しく学ぼう！」という雰囲気
- 星評価やバッジで視覚的に魅力的
- 絵文字で親しみやすさを演出
- ポップで明るいトーン

**こんな方におすすめ:**
- 堅苦しくない、楽しい雰囲気が好み
- 初心者で不安が大きい層
- コミュニティ感を重視したい

## 🔍 共通仕様

### 必須セクション（全パターン共通）
1. **ヒーローセクション**
   - キャッチコピー
   - サービス説明
   - CTAボタン（無料体験）

2. **特徴セクション**
   - 3〜4つの主要メリット
   - アイコン/絵文字で視覚的に

3. **受講プラン比較**
   - 3プラン（¥4,980 / ¥9,800 / ¥19,800）
   - スタンダードプランを強調

4. **受講者の声**
   - 3名のテスティモニアル
   - 実感のあるリアルな声

5. **CTA（行動喚起）**
   - 無料体験への誘導
   - 安心感を与えるメッセージ

### 技術仕様
- **レスポンシブ対応:** 完全対応（モバイルファースト）
- **ファイル形式:** 1ファイル完結（CSS/JSインライン）
- **文字サイズ:** 最小16px、本文18px以上
- **フォント:** システムフォント使用（読み込み不要）
- **アクセシビリティ:** コントラスト比確保、大きめボタン

---

## 📱 対象デバイス

- **スマートフォン:** 最優先（320px〜）
- **タブレット:** 対応（768px〜）
- **デスクトップ:** 対応（1024px〜）

---

## 🚀 使い方

1. 任意のパターンをブラウザで開く
2. レスポンシブ表示を確認（DevTools推奨）
3. 実際のターゲット層にテスト
4. 反応の良いパターンを採用

---

## 🎯 A/Bテスト推奨

3パターンを同時に運用して、以下を計測：
- CTAクリック率
- 滞在時間
- 直帰率
- コンバージョン率

ターゲット層の反応を見て、最適なデザインを選定してください。

## 📝 カスタマイズポイント

### すぐに変更できる箇所
- **色:** CSSの色コード検索置換で一括変更可能
- **文言:** HTML内のテキストを直接編集
- **画像:** アイコン絵文字を画像に差し替え可能
- **プラン内容:** リスト項目の追加・削除が容易

### 注意点
- ボタンのリンク先は現在 `#` または `alert()` です
- 実装時にフォームURLや決済ページへのリンクに変更してください

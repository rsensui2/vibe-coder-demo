# AIエージェント派遣事業 — ソリューション概要

## Ryokoの標準スキルセット（30+スキル）

### 情報収集・分析
| スキル | 説明 | 出力 |
|--------|------|------|
| Tech News Scout | 業界ニュースの自動収集・選別・要約 | Slackレポート + スライドPDF |
| X Bookmark Scout | Xブックマークの分析・カテゴリ分類 | Markdown + スライド |
| Web Research | 競合調査、市場調査、技術調査 | 構造化レポート |
| Lead Scout | リード候補の自動リサーチ | 日次レポート |

### コンテンツ制作
| スキル | 説明 | 出力 |
|--------|------|------|
| Slide Generator | Markdownからプレゼンスライド自動生成 | PPTX + PDF（2K解像度） |
| SNS Content Factory | X投稿30日分 + Instagram図解10枚 | Markdown + PNG |
| Audio Blog | テキストを音声コンテンツに変換 | MP3 |
| Seminar Demo Generator | アイデアから企画〜デモサイトまで全自動 | 50+ファイル |

### 業務自動化
| スキル | 説明 | 出力 |
|--------|------|------|
| Gmail/Calendar連携 | メール要約、スケジュール管理、リマインダー | 朝のブリーフィング |
| 議事録自動整理 | Genspark等の議事録をMarkdown整形 | GitHub + Slack投稿 |
| Webinar Report | 申込データの自動集計・可視化 | 定時レポート |
| Task Management | Linear連携でタスク進捗をリアルタイム管理 | ダッシュボード |

### コミュニケーション
| スキル | 説明 | 出力 |
|--------|------|------|
| Slack統合 | メンション対応、スレッド巡回、リアクション | 自動返信 |
| 問い合わせ返信案 | 問い合わせメールの返信ドラフト自動生成 | Slackスレッド |
| チャンネル配信 | ニュース・レポートの複数チャンネル同時配信 | 14ch同時投稿 |

## 技術アーキテクチャ

```
┌─────────────────────────────────────────────┐
│ 顧客環境（専用VPS）                            │
│                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │ OpenClaw │  │ スキル群  │  │ メモリ   │  │
│  │ Gateway  │──│ (30+)    │──│ MEMORY.md│  │
│  └────┬─────┘  └──────────┘  └──────────┘  │
│       │                                     │
│  ┌────▼─────────────────────────────────┐   │
│  │ チャンネル接続                          │   │
│  │ Slack / Teams / Discord / Email      │   │
│  └──────────────────────────────────────┘   │
└──────────────┬──────────────────────────────┘
               │ API
┌──────────────▼──────────────────────────────┐
│ AI Provider (顧客が直接契約)                  │
│ Anthropic Claude / OpenAI GPT               │
└─────────────────────────────────────────────┘
```

### セキュリティ設計
- **顧客専用VPS**: データは顧客環境に閉じる
- **API直接契約**: AI Providerとの契約は顧客自身（データ漏洩リスクなし）
- **オンプレ対応**: Enterprise向けにオンプレミス構成も提供可能
- **アクセス制御**: オーナー認証、機密情報ポリシー内蔵

## 画面設計（モックアプリ）

### ダッシュボード画面
- 今日の活動サマリー（完了タスク数、生成した成果物数）
- アクティブなエージェント一覧
- 最新の成果物リスト（ワンクリックで開ける）

### タスク管理画面
- Linear風カンバンボード
- リアルタイムステータス更新
- サブエージェントの進捗も可視化

### スキル設定画面
- 有効/無効の切り替え
- cronスケジュール設定
- 通知先チャンネル設定

### 成果物ライブラリ
- 生成したPDF、スライド、レポートを一覧管理
- 日付・カテゴリでフィルタ
- ワンクリックでダウンロード or 共有

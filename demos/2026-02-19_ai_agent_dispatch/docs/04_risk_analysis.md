# AIエージェント派遣事業 — リスク分析

| リスク | 影響度 | 発生確率 | 総合 | 対策 |
|--------|--------|---------|------|------|
| AI精度問題（幻覚、誤情報） | 高 | 中 | 🔴 高 | 人間レビューフロー必須化 |
| API費用の予想外増大 | 中 | 高 | 🔴 高 | 月額上限設定、超過アラート |

## 主要リスクの詳細と対策

### 🔴 AI精度問題（幻覚・誤情報）

**シナリオ**: エージェントが誤った情報を顧客の取引先にメール送信。信用問題に発展。

**対策**:
1. **外部送信は承認制** — メール、SNS投稿、外部API呼び出しは人間の承認を経る
2. **ファクトチェックスキル** — 数値・固有名詞は複数ソースで自動クロスチェック
3. **信頼度スコア表示** — エージェントの回答に信頼度を付与、低い場合は警告
4. **ロールバック機能** — 誤送信時の取り消し・訂正を即座に実行

### 🔴 API費用の予想外増大

**シナリオ**: 大量のサブエージェント起動で月のAPI費用が想定の3倍に。

**対策**:
1. **月額予算アラート** — 80%消化時点で自動通知
2. **スロットリング** — 予算上限到達で低コストモデルに自動切替
3. **コスト最適化** — タスクの複雑さに応じてモデルを自動選択（簡単なタスクはHaiku）
4. **透明な請求** — ダッシュボードでリアルタイムのAPI使用量を可視化

### 🟡 顧客データ漏洩

**対策**:
1. **専用VPS方式** — 顧客データは顧客環境から出ない
2. **API直接契約** — AI Providerへのデータ送信は顧客自身の契約
3. **機密情報ポリシー** — エージェントに内蔵の情報分類・保護ルール
4. **監査ログ** — 全アクション記録、月次セキュリティレポート

### 🟡 競合参入

**現状の競合環境**:
- **SaaS型AIツール**（ChatGPT Team, Copilot）: 機能は豊富だが「自律実行」がない
- **AIコンサル**（アクセンチュア等）: 高額（数千万〜）、カスタム開発に時間がかかる
- **RPA事業者**（UiPath等）: 定型業務のみ、判断を伴うタスクは対応不可

**差別化戦略**:
1. **実績ファースト** — TEKION Group自身が使い倒した実績を最大の営業ツールに
2. **スキル資産** — 30+スキルの蓄積が参入障壁
3. **コミュニティ** — Vibe Coder Bootcamp卒業生をエコシステム化
4. **スピード** — 即日稼働が最大の武器

## コンプライアンス

### AI利用に関する法的考慮

| 領域 | 対応 |
|------|------|
| 個人情報保護法 | 専用VPS方式で顧客環境に閉じる |
| AI事業者ガイドライン | 透明性確保（AIが生成したことを明示） |
| 著作権法 | 生成物の著作権帰属を契約で明確化 |
| 労働者派遣法 | AIエージェントは「派遣」ではなくSaaS提供（法的整理済み） |

### 「派遣」という名称について
- マーケティング上「派遣」と表現するが、法的にはSaaSサービス提供
- 労働者派遣法の適用対象外であることを契約書に明記
- 「AIエージェント利用許諾契約」として締結

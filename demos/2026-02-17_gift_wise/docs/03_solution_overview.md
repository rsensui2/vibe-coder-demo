# Gift Wise — ソリューション概要

## コア機能

### 1. ウィッシュリスト（Wish List）
- **URL貼り付けで自動登録** — ECサイトのURLを貼るだけで商品名・画像・価格を自動取得
- **カテゴリ分類** — 「誕生日」「クリスマス」「いつでも」等のタグ付け
- **優先度設定** — ★1〜3でほしい度合いを表現
- **価格帯フィルタ** — 贈る側が予算に合わせて選べる
- **共有設定** — 全体公開 / 友達限定 / 特定の人のみ

### 2. ギフトログ（Gift Log）
- **もらったギフトを写真付きで記録** — カメラで撮って即登録
- **贈り主・日付・イベントを紐付け** — 「誰から」「いつ」「何の記念日に」
- **お返しステータス管理** — 未対応 / 検討中 / 完了
- **タイムライン表示** — ギフトの歴史を振り返れる

### 3. メッセージカード（Thank You Card）
- **テンプレートから選んでカスタマイズ** — 40種以上のデザイン
- **手書き風フォント対応** — デジタルでも温かみを
- **写真コラージュ** — ギフトの写真+メッセージ
- **SNS/LINEで直接送信** — アプリ内から共有

### 4. リマインダー（Reminder）
- **記念日の自動リマインド** — 2週間前・1週間前・当日に通知
- **お返し期限アラート** — もらってから1ヶ月以内に通知
- **ギフト提案** — 相手のウィッシュリストから予算内の候補を自動提案

### 5. グループギフト（Group Gift）
- **みんなで一つのギフトを贈る** — 割り勘機能付き
- **匿名参加OK** — サプライズを守る
- **進捗共有** — 集金状況・購入状況をリアルタイム表示

## 技術スタック

### フロントエンド
- **React Native** — iOS/Android クロスプラットフォーム
- **Expo** — 開発効率化、OTAアップデート
- **React Navigation** — 画面遷移
- **Reanimated** — スムーズなアニメーション

### バックエンド
- **Supabase** — 認証、データベース、ストレージ、リアルタイム
- **PostgreSQL** — メインDB
- **Edge Functions** — サーバーレスAPI
- **Supabase Storage** — 画像保存

### 外部連携
- **OGP/メタデータ取得API** — URL貼り付けで商品情報自動取得
- **LINE Messaging API** — メッセージカード送信
- **Firebase Cloud Messaging** — プッシュ通知
- **Stripe** — 決済（プレミアム課金・グループギフト集金）

### インフラ
- **Vercel** — Webアプリホスティング
- **Supabase Cloud** — バックエンド
- **Cloudflare** — CDN、画像最適化

## MVP（Minimum Viable Product）

### MVP機能（3ヶ月）
1. ✅ ウィッシュリスト作成・共有（URL自動取得）
2. ✅ ギフトログ（写真撮影・記録）
3. ✅ 友達のリスト閲覧
4. ✅ シンプルなメッセージカード（5テンプレート）
5. ✅ リマインダー（記念日通知）

### Post-MVP（6-12ヶ月）
- グループギフト機能
- EC連携（アフィリエイト）
- AI推薦（贈り物提案）
- 法人向け機能
- 海外展開（多言語対応）

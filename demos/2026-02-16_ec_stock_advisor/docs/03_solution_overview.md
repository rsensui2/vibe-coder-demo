# ソリューション概要 — EC Stock Advisor

## サービス全体像

EC Stock Advisorは、EC業界の上場企業約50銘柄のファンダメンタル・テクニカル・センチメント分析をリアルタイムで提供し、ユーザーの投資スタイルに応じた銘柄提案を行うWebサービスである。

```
[データソース]                    [EC Stock Advisor]              [ユーザー]
├─ 東証API (株価・出来高)    →   ┌─────────────────────┐         ├─ Webダッシュボード
├─ EDINET (決算・有報)       →   │ Data Pipeline       │         ├─ モバイルアプリ
├─ 企業IR (GMV・KPI)        →   │ ↓                   │         ├─ アラート通知
├─ ニュースAPI               →   │ AI Analysis Engine  │    →    ├─ ポートフォリオ
├─ X/SNS (センチメント)      →   │ ↓                   │         └─ レポート
└─ Google Trends             →   │ Recommendation API  │
                                  └─────────────────────┘
```

## 主要機能

### 1. EC銘柄ダッシュボード

トップページに全EC上場企業を一覧表示。業界特化KPIで横断比較できる。

**表示項目:**

| カテゴリ | 指標 | 説明 |
|---------|------|------|
| 株価 | 現在値、変動率、52週レンジ | リアルタイム（15分ディレイ） |
| バリュエーション | PER、PBR、PSR、EV/EBITDA | セクター平均との比較付き |
| EC固有KPI | GMV（流通取引総額） | 四半期ごとの推移グラフ |
| | テイクレート | GMV対売上高比率 |
| | AOV（平均注文単価） | 商材カテゴリ別 |
| | アクティブバイヤー数 | 開示企業のみ |
| センチメント | ニューススコア | AI分析による好悪材料判定 |
| | SNS言及量 | X、Yahoo!ファイナンス掲示板 |
| トレンドスコア | 総合スコア（0-100） | 独自アルゴリズムで算出 |

### 2. トレンドスコアリングエンジン

独自の「ECトレンドスコア」を各銘柄に付与。以下の5要素を重み付け算出：

```
ECトレンドスコア = Σ(Wi × Si)

W1: 業績モメンタム（30%）  → 売上成長率、営業利益率の推移
W2: GMV成長率（25%）       → EC固有の成長指標
W3: バリュエーション（20%） → PSR、PEGレシオの相対評価
W4: センチメント（15%）     → ニュース・SNSのポジネガ分析
W5: テクニカル（10%）       → 移動平均線、RSI、出来高トレンド
```

**スコア分類:**
- 🔥 80-100: 強気推奨（BUY）
- 📈 60-79: やや強気（WATCH）
- ➡️ 40-59: 中立（HOLD）
- 📉 20-39: やや弱気（CAUTION）
- ⚠️ 0-19: 弱気（SELL検討）

### 3. AI投資提案（Premiumプラン）

ユーザーの投資スタイル・リスク許容度を入力すると、AIがEC銘柄でポートフォリオを構成。

**投資スタイルプリセット:**

| スタイル | 特徴 | 想定ポートフォリオ例 |
|---------|------|-------------------|
| 成長重視 | 高GMV成長率銘柄中心 | メルカリ30%、BASE20%、Hamee15%... |
| 安定重視 | 大型+高配当 | 楽天G25%、ZOZO25%、MonotaRO20%... |
| バリュー狙い | 割安放置銘柄 | BUYSELL30%、Hamee25%... |
| テーマ投資 | 特定テーマ集中 | 「越境EC」「ライブコマース」等テーマ別 |

### 4. リアルタイムアラート

| アラート種別 | 内容 | 配信方法 |
|-------------|------|---------|
| 決算速報 | 決算発表後5分以内にAI要約配信 | Push/メール/LINE |
| 株価急変動 | ±5%以上の変動を検知 | Push |
| トレンドスコア変動 | スコアが10pt以上変動 | 週次ダイジェスト |
| 業界ニュース | EC業界の大型ニュース | Push/メール |
| 規制変更 | 景表法、特商法等の改正情報 | メール |

### 5. 業界レポート（月次）

- EC業界全体の月次レビュー
- 注目銘柄ピックアップ（3-5銘柄）
- セクターローテーション分析
- 海外EC企業との比較
- ゲストアナリストコラム

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | Next.js 15 + TypeScript + Tailwind CSS |
| バックエンド | Python (FastAPI) + Node.js |
| データベース | PostgreSQL + TimescaleDB（時系列データ） |
| キャッシュ | Redis |
| AI/ML | OpenAI API (GPT-4o) + 自社ファインチューニングモデル |
| データパイプライン | Apache Airflow + dbt |
| インフラ | AWS (ECS, RDS, ElastiCache, S3, CloudFront) |
| 監視 | Datadog + Sentry |
| CI/CD | GitHub Actions |

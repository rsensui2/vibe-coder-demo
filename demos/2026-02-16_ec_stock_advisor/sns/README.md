## 🐦 X（Twitter）投稿ガイド

### 投稿カレンダー
`x_posts_30days.md` に30日分の投稿文を収録。各投稿は140文字以内、ハッシュタグ・絵文字付き。

### 曜日別テーマ
| 曜日 | テーマ | 内容 |
|------|--------|------|
| **月曜日** | 市場分析 | EC市場全体のトレンド、成長予測、業界動向 |
| **火曜日** | 銘柄紹介 | 注目銘柄・隠れた優良株の分析とトレンドスコア |
| **水曜日** | EC業界ニュース | 最新ニュース、技術革新、M&A情報 |
| **木曜日** | 投資Tips | 投資テクニック、リスク管理、ツール活用法 |
| **金曜日** | 週間まとめ | 今週の市場サマリー、来週の注目ポイント |
| **土曜日** | 学習コンテンツ | 分析手法、財務諸表の読み方、解説記事 |
| **日曜日** | モチベーション | 投資マインド、資産形成の意義、励まし |

### 推奨投稿時間
- **平日**
  - 朝: 7:00-8:00（通勤時間帯）
  - 昼: 12:00-13:00（ランチタイム）
  - 夜: 20:00-21:00（リラックスタイム）
- **土日**
  - 午前: 10:00-11:00
  - 午後: 15:00-16:00

### ハッシュタグ戦略
- **必須タグ**: `#EC投資` `#株式投資`（全投稿に含める）
- **曜日別タグ**: 各テーマに応じて2-3個追加
- **トレンドタグ**: 時事ニュースに応じて柔軟に調整

### エンゲージメント向上Tips
- 絵文字を適度に活用（各投稿2-3個）
- 疑問形や呼びかけで関心を引く
- データ・数字を具体的に提示
- 画像・グラフと併用でインプレッション増加

## 📞 サポート・問い合わせ対応

### よくある質問への対応テンプレート

**Q: 無料プランでどこまで使えますか？**
A: 基本的な市場分析とトレンドスコア閲覧が可能です。詳細な銘柄分析やAI投資提案はスタンダードプラン以上をご利用ください。14日間無料トライアルでお試しいただけます！

**Q: データの信頼性は？**
A: 上場企業の公式IR情報・金融庁開示資料・主要EC業界レポートを基に分析しています。AIによる予測は過去データに基づく参考情報としてご活用ください。

**Q: 投資初心者でも使えますか？**
A: はい！直感的なUIと分かりやすい解説で、初心者の方でも安心してご利用いただけます。学習コンテンツも豊富にご用意しています。

# 市場調査 — Pre-Onboarding Mission

## 日本の中途採用市場 概況

### 市場規模と成長トレンド

| 指標 | 2022年 | 2023年 | 2024年 | 2025年(推計) |
|------|--------|--------|--------|-------------|
| 中途採用者数 | 328万人 | 342万人 | 351万人 | 365万人 |
| 人材紹介市場規模 | ¥3,080億 | ¥3,420億 | ¥3,800億 | ¥4,100億 |
| 求人倍率（中途） | 1.82 | 2.01 | 2.15 | 2.25 |
| 転職成功者の平均年齢 | 31.7歳 | 32.0歳 | 32.4歳 | 32.8歳 |

*出典: 厚生労働省「雇用動向調査」、リクルートワークス研究所*

### 中途採用を取り巻く変化

1. **人手不足の深刻化**: 有効求人倍率は1.3倍超を継続。特にIT・DX人材は4倍以上
2. **転職の一般化**: 35歳以上の転職が増加。「転職35歳限界説」は完全に崩壊
3. **採用コストの高騰**: 人材紹介フィー（年収の30-35%）が標準。1人あたり100万円超
4. **オンボーディングへの注目**: SHRM調査で「体系的オンボーディングで新入社員の定着率が82%向上」が広く知られるように
5. **リモートワークの定着**: 入社前のオンラインコミュニケーションのニーズが増加

## 競合分析

### 直接競合（入社前オンボーディングに特化）

**日本市場に直接競合はほぼ存在しない。** これはブルーオーシャンであると同時にリスク（市場教育が必要）でもある。

| サービス | 国 | 特徴 | 弱み |
|---------|-----|------|------|
| Enboarder | 豪 | ワークフロー型プレボーディング | 日本未対応、AI対話なし |
| Sapling (Kallidus) | 米 | HRIS統合型オンボーディング | エンタープライズ向け、日本語なし |
| Talmundo | ベルギー | ゲーミフィケーション型 | 日本市場なし |
| Click Boarding | 米 | コンプライアンス重視 | 事務手続き中心でミッション設計なし |

### 間接競合（日本のHRテック）

| サービス | カテゴリ | 入社前対応 | 脅威度 |
|---------|---------|-----------|-------|
| SmartHR | 労務管理 | 入社手続きの電子化のみ | 中（機能拡張の可能性） |
| HRMOS | 採用管理 | 内定者フォロー機能あり | 高（最も近い位置） |
| カオナビ | タレントマネジメント | なし | 低 |
| Wantedly | 採用/エンゲージメント | 企業文化の発信 | 中（文化面で競合） |
| TUNAG | エンゲージメント | 入社後の施策のみ | 低 |
| モチベーションクラウド | サーベイ | 入社後の計測のみ | 低 |

### 競合ポジショニングマップ

```
AI活用度 高

    │        ★ Pre-Onboarding Mission
    │          （AI対話 × 入社前特化）
    │
```

## ペルソナ

### ペルソナ1: 山田太郎（転職者）

| 項目 | 詳細 |
|------|------|
| 年齢 | 32歳 |
| 現職 | 大手メーカー 営業企画（5年目） |
| 転職先 | SaaS企業 事業開発マネージャー |
| 入社まで | 2ヶ月 |
| 状況 | 現職の引き継ぎで忙しく、転職先のことを考える余裕がない |
| 不安 | 「SaaS業界は初めて。ちゃんとやれるかな」「チームに馴染めるかな」 |
| ニーズ | 入社前に自分の立ち位置を理解し、Day1に自信を持って臨みたい |
| 利用シーン | 通勤電車の中、就寝前の15分 |

**カスタマージャーニー:**
1. 内定承諾 → 企業からPre-Onboarding Missionの招待メール受信
2. スマホでアカウント作成、ウェルカム動画視聴
3. 通勤中にAI対話でPhase 1（自己発見）を完了
4. 週末にカルチャーコンテンツ（社員インタビュー3本）を視聴
5. 翌週にPhase 2-3を完了、ミッションステートメント生成
6. 入社前に上司にSlackで共有「私のミッションは○○です」
7. 入社初日、自己紹介で堂々とミッションを宣言

## 参入タイミング分析

### なぜ「今」なのか

| 要因 | 詳細 | タイミングの良さ |
|------|------|-----------------|
| AI対話技術の成熟 | Claude 4, GPT-5クラスで自然な日本語対話が可能に | ⭐⭐⭐ |
| 人手不足の深刻化 | 採用した人材の定着が最重要課題に | ⭐⭐⭐ |
| オンボーディング市場の黎明期 | 日本ではまだ体系的なプレボーディングがほぼない | ⭐⭐⭐ |
| HRテックSaaS市場の拡大 | 2025年HRテック市場¥1,200億超、年成長15% | ⭐⭐ |
| 働き方改革の浸透 | 企業が「人的資本経営」を意識し始めた | ⭐⭐ |

### タイミングリスク

| リスク | 可能性 | 対策 |
|--------|--------|------|
| 早すぎる（市場教育コストが高い） | 30% | セミナー・コンテンツマーケで啓蒙 |
| 大手が先に参入 | 10% | 現時点で同等サービスの開発情報なし |
| AI規制で制約 | 15% | 「評価」でなく「支援」の位置付け |

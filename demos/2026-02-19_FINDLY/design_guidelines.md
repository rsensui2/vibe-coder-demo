# FINDLY デザインガイドライン

## ブランドカラー

```
メイン:    #1A6FDB（ロイヤルブルー）— 信頼・技術・鉄道の青
アクセント: #00C9A7（ミントシアン）— 発見・安心・通知の瞬間
背景dark:  #0A0F1E（深紺）— 夜の都市・精密技術
背景light: #F4F7FF（薄青白）— クリーン・安心感
テキスト: #FFFFFF / #1A1A2E
グラデーション: linear-gradient(135deg, #1A6FDB 0%, #00C9A7 100%)
エラー:   #FF4757
成功:    #00C9A7
```

## タイポグラフィ

```
見出し: "Noto Sans JP" Bold / 英語部分 "Inter" Bold
本文:   "Noto Sans JP" Regular
モノスペース: "Fira Code"（技術仕様・コード）
```

## ビジュアルスタイル

- **ダーク×ブルー基調**: 夜の駅ホーム、都市の鉄道ネットワークをイメージ
- **グロウエフェクト**: BLE電波の広がりをシアングロウで表現
- **アニメーション**: 通知が届く瞬間のパルスエフェクト（ミントシアン）
- **アイコン**: 線描きスタイル、角丸なし（技術的・精密なイメージ）
- **写真**: 都市・鉄道・スマートフォン。人物は自然体（演技的NG）
- **レイアウト**: カード型、余白大きめ、グリッド厳守

## UIパターン

- ボタン: 角丸8px、メイン色（グラデあり）またはアウトライン
- カード: bg rgba(255,255,255,0.05) / border rgba(0,201,167,0.2) / backdrop-blur
- バッジ: ミントシアン塗り、ロイヤルブルーテキスト
- 地図ビジュアル: ダーク地図上にシアンのピン・パルスエフェクト

## コピートーン

- **キャッチコピー**: 「見つかる前提で、生きる。」
- **サブコピー**: 「鉄道インフラと直結した、次世代落とし物防止タグ」
- **トーン**: 信頼感×テクノロジーの誠実さ×「安心」の温かさ
- 過度な感嘆符NG、落ち着いたプロフェッショナル
- 数値を積極的に使う（23秒、400万件、72%）

## ロゴ使用

- FINDLYロゴ: "FIND" ホワイト + "LY" シアン
- タグマーク: 菱形にBLE電波rippleデザイン
- 背景暗い場合: フルカラー / 背景明るい場合: ネイビー単色

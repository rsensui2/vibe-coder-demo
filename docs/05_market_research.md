# 市場調査

## 日本の自動車市場概況（2025年データ）

### 販売台数
| 区分 | 年間台数 | 前年比 |
|------|---------|--------|
| 新車（登録車） | 約290万台 | -2.1% |
| 新車（軽自動車） | 約160万台 | +1.3% |
| 中古車 | 約650万台 | +0.5% |

**中古車市場は新車市場の約1.4倍の規模。**

### 平均購入価格
- 新車平均: 約370万円（軽除く）
- 中古車平均: 約175万円
- 中古車のうち「高年式・高価格帯（200万円以上）」: 全体の約35%

### 購買行動の変化
- Web検索で車を探す人: **78%**（2020年: 62%）
- 購入前にオンラインで比較検討する平均期間: **2.3ヶ月**
- 「新車と中古車で迷った経験がある」: **64%**（カーセンサー調査）

## 競合分析

### 直接競合（新車 vs 中古車 比較サービス）
**→ 現時点で直接競合は存在しない。** これが最大のチャンス。

### 間接競合

| サービス | 強み | 弱み | CarScaleとの違い |
|---------|------|------|----------------|
| **カーセンサー** | 圧倒的な中古車在庫数 | 新車との比較不可 | 中古車検索に特化、横断比較なし |
| **グーネット** | 詳細な車両情報 | UIが複雑 | 同上 |
| **価格.com 自動車** | 新車スペック比較が充実 | 中古車は外部リンクのみ | 新車同士の比較に特化 |
| **MOTA** | 新車見積もり一括依頼 | 中古車は別サービス | 購入プロセス寄り、比較体験なし |
| **ガリバー** | 買取・販売の両面 | 自社在庫のみ | 販売チャネル、比較ツールではない |

### ポジショニングマップ
```
         新車に強い
            │
    価格.com ●    ● MOTA
            │
 情報重視 ──┼── 体験重視
            │
  グーネット ●    ★ CarScale
            │      （ここにポジション）
 カーセンサー●
            │
         中古車に強い
```

**CarScaleは「新車×中古車の横断」かつ「体験重視（直感的な比較UI）」というブルーオーシャンに位置する。**

## 市場機会

### 狙うべきセグメント
- **予算300〜600万円帯**: 新車と中古車の車格差が最も顕著に出る価格帯
- **SUV/ミニバン検討層**: 家族持ちの買い替え需要が最も大きい
- **初めての輸入車検討層**: 「中古なら外車に手が届く」という気づきを提供

### 参入タイミング
- 中古車のオンライン購買率は年々上昇中
- 半導体不足が解消し、新車供給が正常化 → 中古車価格も安定化
- **今がベストタイミング**
